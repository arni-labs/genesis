How TemperPaw Uses Genesis

  • TemperPaw resolves a pinned Genesis app ref like owner/app@hash.
  • It calls the spec-owned OData action POST /tdata/Apps('app-owner-app')/App.Install with the target tenant.
  • The equivalent CLI is temper install owner/app@hash --tenant default --url https://genesis.example.
  • Only helper skills remain as skills; Temper apps and agent skills inside those apps are loaded from installed Genesis app bundles.

Install Surfaces

OData POST /tdata/Apps('app-owner-name')/App.Install
Temper CLI temper install owner/name@hash --tenant default --url https://genesis.example
TemperPaw tool install_app({"source":"genesis","app_ref":"owner/name@hash","tenant":"default","url":"https://genesis.example"})
